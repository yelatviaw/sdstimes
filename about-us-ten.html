About The SDS Times

The SDS Times is a one-man news website established in July 2023 at the second week of Correlation-One/Amazon software development (front-end) training program. I am building this website for practice purposes, not to form a media company called SDS Times.The main reason for taking a front-end software development class is to put together basic knowledge of HTML and CSS that I collected from different sources in a structured form in order to qualify for the entry level job markets and further skills.

On the HTML topics of the web development course, we learned how to link an image, video, and audio, unordered and ordered lists, favicons, imagemap and where to get free contents that help the process of building websites like Unsplash and Free Music Archives. In HTML, everything from the start tag to the end tag is called element. Examples of elements are html, title, body, center, form, table, label, select, input, output, button, div, br, p, and h. The element called html is the boss of all elements and dictates the whole HTML page.

Attribute gives additional info about HTML elements, for example, src attribute shows the path of the image to displayed, href attribute the URL of the page, alt attribute tells the alternate text of an image used on the website for different reasons. Attributes are always specified in the beginning of a tag and more examples of attributes are action, target, method, for, id, and name. HTML Style element gives additional modification to an element such as text color, size of font, padding, border, and margin. It is used to modify the background of webpage's body, header, or footer. In addition to that, to resize text and to change the position of text as alignment we can use HTML styling. Favicon is a tiny image next to the title of the website. Input element gives with type attribute to show radio button, checkbox, text field, clickable button, submit button. The training will continue and for now I linked one story about Correlation-One that was posted on LinkedIn in 2022.
